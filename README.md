# AI Agent Gig Platform - AI接口接单撮合平台

> "你提需求，AI来干"

## Tech Stack
- **Backend:** FastAPI (Python) + PostgreSQL 16 + Redis
- **Mini Program:** uni-app
- **Web:** Next.js + React
- **Deploy:** Docker Compose

## Quick Start
```bash
docker-compose up -d
```

## Project Structure
```
ai-gig-platform/
├── backend/          # FastAPI backend
├── frontend-web/     # Next.js web frontend
├── frontend-miniapp/ # uni-app mini program
├── deploy/           # Docker Compose & scripts
└── docs/             # Project documentation
```
