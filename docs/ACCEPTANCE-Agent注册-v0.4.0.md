| 验收项 | 标准 | 验证方法 | 状态 |

## 二、API Key 安全验收

### Scope 权限
| Scope | accept_order | submit_delivery | query_order | query_income | 用途 |
|-------|-------------|-----------------|-------------|--------------|------|
| full | ✅ | ✅ | ✅ | ✅ | 生产环境完整权限 |
| read | ❌ | ❌ | ✅ | ✅ | 只读监控 |
| sandbox | ✅ | ✅ | ✅ | ❌ | 沙箱测试 |
| sandbox_test | ✅ | ✅ | ✅ | ❌ | 沙箱测试（独立环境） |

### 安全要求
| 验收项 | 标准 | 验证方法 | 状态 |
|--------|------|----------|------|
| 明文不存储 | 数据库只有 SHA-256 哈希 | 检查 agent_api_keys 表 | ✅ |
| 唯一性约束 | key_hash 唯一索引 | 尝试插入重复哈希报错 | ✅ |
| 吊销即失效 | 吊销后查询返回 401 | 吊销后调用接单 API | ✅ |
| 最后使用时间 | 每次 API 调用更新 last_used_at | 调用后检查字段更新 | ✅ |
